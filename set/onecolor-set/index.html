注意事項

特色印刷（シルバー、ホワイト、蛍光ピンク）について
  • 特色（シルバー、ホワイト、蛍光ピンク）をご利用の場合、グレースケールまたはモノクロ2階調のデータでご入稿ください。黒色で作成されたRGBやCMYKデータでご入稿された場合は、弊社にてグレースケールに変換して進めさせていただきます。
  • 推奨解像度は600dpiになります。
  • シルバー印刷で非コート紙などの用紙を使用される場合は、メタリック感が目立ちにくくなります。
  • シルバー印刷は線が少し太り、白抜きが少し細く潰れやすくなります。
  • 特色は用紙の持つ色に多少影響されます。
  • 特色と用紙の組み合わせによっては印刷が見えずらくなったり、色が変わる可能性がございます。
  • 作成データで特色（シルバー、ホワイト、蛍光ピンク）印刷の仕上がりを確認されたい場合、「テスト印刷（試し刷り）」（有料）をお勧めしています。用紙や枚数でお値段が変わります。また、大イベント期間中などはお受けできない場合もございますので、まずはお気軽にお問い合わせください。
見え辛くなる組み合わせについて
  • ホワイト印刷と淡色または白い用紙
  • シルバー印刷と両更クラフト
  • 蛍光ピンク印刷と濃色用紙
  • カラー印刷と黒い用紙（白押さえがございません）
    などの組み合わせは、視認性が低くなるためオススメしません。特にホワイト印刷は白い用紙に印刷すると、光の当たる角度と見る角度によっては一切見えなくなる場合がございます。本文ホワイト印刷をご利用の場合は「色上質 厚口 黒」をご選択ください。

オンデマンド印刷について

オンデマンド印刷はオフセット印刷のような版を使った印刷とは異なり、トナーでの印刷となります。そのため、以下のようなオンデマンド印刷特有の現象が生じる場合がございます。
・色や濃さなどが統一化されていない。
・面積の広いベタ塗りの部分や平網の部分にムラが出る。
・紙に吹きつけたトナーを200度近い高熱で定着させるため、紙に含まれた水分が奪われ、紙が波打つ。
・本の背などの折り部分の印刷は、トナーがはがれやすい。
・黒ベタの多い絵柄の場合、稀に白抜けが発生する。
・稀にトナーのカスが付着することがある。
・オンデマンド印刷でPP加工をされた場合、断裁位置に濃いベタ面があるときは、PPフィルムの剥がれが生じやすくなることがある。
・平網が広範囲の場合、用紙ロットによってキズが入りやすいことがある。（アートポストPP加工推奨）
・平網が広範囲の場合、トナーの斑点が発生しやすい。
・トナーの擦れによる汚れが起こりえそうなご本では梱包時に本の帯留めや紙の挟み込みでトナー汚れ、トナーの色移りの予防を出来る限り行っております。ご自身で持ち運び、出荷移動の際もお取り扱いにはご注意ください。
