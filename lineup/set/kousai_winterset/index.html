加工について

エンボス紙、ラフ紙の加工についてのご注意
エンボス紙、ラフ紙は凹凸の影響により、加工ムラが起こりやすい特殊紙となります。PP加工をされる場合、フィルムの気泡の目立ちや剥がれが起きやすくなります。箔押し加工は用紙の柄の影響を受けやすく、Brilliant箔など一部質感がわかりにくくなります。細かい部分の抜き不良、潰れなどが起きやすくなります。PP加工+箔押しをする場合は箔の定着不良がより起きやすくなるため、併用はおすすめしません。
ご使用される場合は、上記の点にご了承の上ご注文ください。

PP加工についてのご注意
・クリアPP加工を行った場合、赤みが多少強くなり全体的に濃度が少し濃くなります。
・エンボス紙にもPP加工は可能ですが、凹凸の影響で気泡が入り剝がれやすくなる可能性がございます。また特定の特殊紙とPP加工と絵柄の組み合わせによってはPP加工が困難になる場合がございます。その際はご連絡させていただきます。
・PP加工によって特殊紙本来の質感が失われる場合がございます。
・PP加工はフィルムと紙の伸縮性の違いにより、表紙の紙が反ってしまう場合がございます。紙が柔らかく、用紙の厚みが薄い（120㎏～135㎏）場合は、紙の反りが起こりやすくなります。紙厚は170㎏以上が推奨です。

箔押し・空押し加工についてのご注意（共通）
・細かすぎる線や点は潰れてしまう可能性がありますので、1ptまたは3px以上の線で作成してください。
・データは黒（Ｋ100％）のみで作成して下さい。グレイ部分（Ｋ1～99％）部分は、50％を境に0％もしくは100％に変換されます。
・1枚ずつ手作業での加工となるため、どうしても多少のズレが発生してしまいます。ご了承くださいませ。
・エンボス用紙にPP加工+箔押しをする場合は箔の定着不良がより起きやすくなるため、併用はおすすめしません。また、箔押しと空押しが重ならない場合でも、距離が近い、または隣接する場合は、フチの部分で綺麗な仕上がりとならないことがございます。

箔押しについてのご注意

▲細かい凹凸があるエンボス紙、ラフ紙は、箔のかすれ、潰れが発生しやすくなります。
特にベタ面が広い箇所でカスレが発生しやすく、また、細かい白抜きがうまく再現できない事がありますので、白抜き部分は太めに作成して下さい。

▲パール箔は細かい部分や白抜きが苦手な箔になります。特にエンボス紙、ラフ紙は上記の図のように抜ききれない部分が発生する事があります。

  • ベタ面に小さな気泡（穴）が入る場合もございます。
  • 白抜き文字や絵柄は、デザインや用紙により細かい部分が抜けきらずに潰れが発生することがございます。
  • 凹凸が強い用紙は、箔がうまく定着せずに抜けが発生する場合がございます。（特に凹部分）
  • 箔押しは背にかかるデザインの場合、箔が割れたり剥がれる場合がございますのでおすすめしません。
  • エンボス紙にPP加工した用紙に箔を押す場合、用紙や箔の種類によってはうまく付かない組み合わせがございます。その場合加工や用紙変更のお願いのご連絡をさせていただく事がありますのでご了承ください。
  • Bタイプ箔をご利用の場合、PP加工との併用はできません。
  • Bタイプ箔はメタルインキの上では定着不良が発生します。箔とメタルインキの印刷が被らないデザインをおすすめします。その他の箔も定着がしにくい為、メタルインキのベタ面の上の箔デザインは避けていただくようお願いいたします。デザインによっては修正、箔の変更をお願いする場合がございますのでご了承下さい。
  • ホログラム箔、リフレックス箔に関しては材料の性質上、模様の繋ぎ目部分(5～8mm程度)が稀に出てしまう場合があります。予めご了承ください。
  • オフセットフルカラー印刷の場合、濃色の上に透明箔を押すと定着不良が起きやすくなります。（透明、パール、ホログラム ドット透明 HK204、リフレックス透明 HK038）濃色と透明箔が被らないデザインをおすすめします。

空押し加工についてのご注意

  • 背や断ち切りにかかるデザインも可能ですが、断面にスジがはいる可能性がございます。
  • 160kg以上の用紙、コート紙やPP加工に対しては、用紙によって凹み具合が目立たない場合がございます。OKフロートやOKムーンカラーなどの用紙の使用をおすすめいたします。
  • OKフロートやOKムーンカラーの空押しは、加工部分の紙の色が変わる加工です。加工部分に印刷のベタ面や塗りなどがあると紙の色の変化はほとんどわかりません。
  • オンデマンド印刷の上に加工を行った場合、トナーが溶けて黒ずんだり剥がれる場合がございます。オンデマンド印刷の場合は絵柄の無い白場部分への加工をおすすめいたします。
  • 空押しとPP加工の併用は空押しが目立たない仕上がりになります。

濃色用紙へのオンデマンドカラー印刷は白トナーによる白押さえがない状態で印刷するため、用紙と印刷するデータの色の組み合わせによっては非常に見え辛くなる可能性があります。

インキについてのご注意
・メタルインキのみ不透明インキとなります。（※多少下地の影響を受けます）
・２色刷りでお使いいただく場合、必ず使用インキの順番を明記して下さい。
・メタルインキを２色目にすると、先に印刷されたインキは隠れてしまいます。
・透明インキを２色目にすると、下のインキの色の影響を受けます。（乗算レイヤーのようなイメージになります）
・透明インキは紙の色の影響を受けます。
