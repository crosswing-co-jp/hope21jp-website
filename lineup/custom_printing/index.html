カスタムセット詳細

部数

オフセット印刷：50部～10,000部
オンデマンド印刷：10部～500部

オフセット印刷で50部以下ご希望の場合、50部の料金になります。

オンデマンド印刷で10部以下ご希望の場合、10部の料金になります。

ページ数

中綴じ：表紙込み12～28ページ
無線綴じ：表紙込み20～500ページ

綴じ種類についてはこちら

無線綴じ20ページ以下をご希望の場合、表紙込み20ページの料金になります。

8ページの場合は折り本になります。

サイズ

B5 / A5 / B6 / 新書 / 文庫 / A4

綴じ

無線綴じ / 中綴じ / 中綴じ（共紙） / 折り本（8ページ）

中綴じ、折り本は表紙込み28ページ以下（4ページ単位）限定になります。

中綴じ（共紙）はオフセット印刷50部からご利用できます。

入稿形式

データ入稿 / アナログ入稿

アナログ入稿の場合、納期が「＋3日営業日」追加されます。

印刷

綴じ種類

無線綴じ
背を糊で固めて、表紙でくるむように綴じる製本です。
ページ数の多い、マンガや小説などの同人誌にも最適です。
また無線綴じにすることで印刷物の背に文字を入れることができるようになります。

中綴じ
二つ折りにした紙の折り目の部分を針金で留めます。無線綴じと違い、背にあたる部分がありません。
表紙と本文紙に同じ紙を使用する共紙中綴じ本がポピュラーです。

共紙中綴じ本

※8ページの場合は折り本になります。
